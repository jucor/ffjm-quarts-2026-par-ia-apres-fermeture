# Probleme 5 : NOEL A TROP PAR RAPPORT A LEON (coefficient 5)

## Enonce du probleme

Noel avait **2026** mathsous ; Leon n'en avait que **206**.
Noel a donne des mathsous a Leon, mais il lui en reste encore **deux fois plus** que Leon.

**Combien Noel a-t-il donne de mathsous a Leon ?**

## Resolution

### Mise en equation

Soit **x** le nombre de mathsous que Noel a donnes a Leon.

Apres le transfert :

- Noel possede : **2026 - x** mathsous
- Leon possede : **206 + x** mathsous

La condition du probleme indique qu'apres le don, il reste a Noel **deux fois plus** de mathsous que Leon. On traduit cela par l'equation :

$$2026 - x = 2 \times (206 + x)$$

### Resolution de l'equation

On developpe le membre de droite :

$$2026 - x = 412 + 2x$$

On regroupe les termes en x a droite et les constantes a gauche :

$$2026 - 412 = 2x + x$$

$$1614 = 3x$$

On divise les deux membres par 3 :

$$x = \frac{1614}{3} = 538$$

## Verification

Apres que Noel a donne 538 mathsous a Leon :

- Noel possede : 2026 - 538 = **1488** mathsous
- Leon possede : 206 + 538 = **744** mathsous

On verifie bien que : 1488 = 2 x 744

La condition est satisfaite.

On peut egalement verifier que le total est conserve :
- Avant : 2026 + 206 = **2232** mathsous
- Apres : 1488 + 744 = **2232** mathsous

## Reponse

**Noel a donne 538 mathsous a Leon.**
